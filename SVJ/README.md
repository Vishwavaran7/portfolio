> **SVJ Corporate Directory** is a high-performance, enterprise-grade client and corporate registry portal built for **SVJ Lee Asia Corporate Consultants Limited**. It features an elegant glassmorphism dark-mode UI with gold accents, dynamic search and filtering, and centralized access to company registration, compliance, and corporate client documents.

---

## 🌟 Key Features

*   🔍 **Dynamic Instant Search:** Client and company directory database with real-time fuzzy search capabilities, allowing consultants to filter records instantly by company name, CR number, or directors.
*   🗂️ **Centralized Document Vault:** Direct local links to standard regulatory corporate forms and templates for client onboarding:
    *   **KYC & CDD Profiles:** `KYC FORMAT.docx`, `CD FORM.docx`, and `CDD FORM NEW.docx`
    *   **Regulatory Return Templates:** `ND4 Company Name.xlsx` (Shares registration) and `NDR1` sheets
    *   **Company Secretarial (SCR):** `SCR CLASSICPLUS.docx` and `SCR MART.docx`
*   💎 **Premium Glassmorphic Enterprise UI:**
    *   **Branding & Colors:** Deep slate blue background (`#020817`) paired with executive gold highlights (`#D4AF37`) and frosted-glass panels.
    *   **Typography:** Editorial type-hierarchy utilizing Cormorant Garamond, Playfair Display, and Noto Sans HK (supporting English and Traditional Chinese characters).
*   📱 **Responsive Mobile-First Architecture:** Fluid responsive layout scaling seamlessly from desktop widescreen panels to mobile screens.

## 🚀 How to Run Locally

Since the SVJ Corporate Directory is a static HTML app, you can launch it immediately:
1. Double-click on `SVJ LEE.html` to open it in any web browser.
2. Alternatively, run a local Python HTTP server from the `portfolio` root directory:
   ```bash
   python -m http.server 8000
   ```
   Then visit `http://localhost:8000/SVJ/SVJ%20LEE.html`.
